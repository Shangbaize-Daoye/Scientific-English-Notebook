## 科学型英语单词、句子助记单词本（开发中）

用于平常积累英语词汇、句子，并科学助记。

## 使用方法

首先要有Python环境，并pip安装一些依赖包。

Windows环境下可以直接运行launch.exe，其他环境要解释执行notebook.py。

* 按键x：笔记本状态下切换词汇、句子，浏览保存词条页面下，也用于切换词汇、句子。
* 按键t：切换积累和保存的页面。
* 按键s：保存词条（词汇或句子），只在积累页面生效。
* 按键q：退出，并同时保存当前浏览进度。
* 上下左右健：上一条或下一条。（n、p键也支持！）
* Ctrl + p：切换暂停
